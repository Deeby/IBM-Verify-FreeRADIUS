# IBM Verify SDK for C

## Quick start
ISAM Verify SDK requires [libcurl](https://curl.haxx.se/libcurl/c/libcurl.html) and [cJSON](https://github.com/DaveGamble/cJSON)
```
#include <curl/curl.h>
#include "cJSON.h"
#include "isam.c"
```

## Interfaces 

ISAM_CALL_AUTH_POLICY(policy);

## Policy response object methods

- Headers
- Code
- Success
- Body
